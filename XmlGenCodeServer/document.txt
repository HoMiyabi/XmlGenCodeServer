服务器开发文档。
使用C# ASP.NET Core实现Http服务器。
数据库使用sqlite，使用ORM，数据就保存到程序下的data目录。
使用最现代的开发方式，自动创建data目录，数据库存储原始文本。

数据表:
file:
    id: int 主键自增
    file_name: varchar 文件名，添加普通索引
    file_content: text 文件内容，文本

接口，接口参数都通过json格式传递，返回使用json格式返回，统一Result{Code, Msg, Data}:

传输过程中FileContent使用base64编码，因为含有特殊符号

获取所有文件列表:
    POST /get_all_files
    返回:
    {
        "Code": 0,
        "Msg": "success"
        "Data":
        [
            {
                "Id": 1，
                "FileName": "文件名",
                "FileContent": "文件内容"
            }
        ],
    }

创建文件:
    POST /create_file
    返回 Data:
    {
        "Id": 1,
        "FileName": "新程序_{id}",
        "FileContent": "文件内容" // 新文件为空
    }

删除文件:
    POST /delete_file
    参数:
    {
        "Id": 1
    }
    返回 Data: null

复制文件:
    POST /duplicate_file
    参数:
    {
        "Id":, // 源文件id
    }
    返回 Data:
    {
        "Id":, // 新文件id
        "FileName": "新文件名", // 文件名从"{原文件名}_{i}", i = 1开始尝试，直到不重名
        "FileContent": "文件内容"
    }

修改文件名:
    POST /update_file_name
    参数:
    {
        "Id": 1,
        "FileName": "新文件名",
    }
    返回 Data: null

修改文件内容:
    POST /update_file_content
    参数:
    {
        "Id": 1,
        "FileContent": "新文件内容"
    }
    返回 Data: null
